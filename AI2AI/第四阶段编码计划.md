## 二、第四阶段任务分解

### 第15周：数据分析体系建设（上）

#### T4.15.1 数据分析基础设施
| 任务 | 文件 | 说明 |
|------|------|------|
| ETL服务 | `modules/analytics/etl.service.ts` | 数据抽取/转换/加载 |
| 聚合表Entity | `modules/analytics/entities/*.entity.ts` | 日/周/月聚合表 |
| 定时任务 | `modules/analytics/analytics.cron.ts` | 每日/每周/每月聚合 |
| 分析模块 | `modules/analytics/analytics.module.ts` | 模块注册 |

#### T4.15.2 用户RFM分析
| 方法 | 功能 |
|------|------|
| calculateRFM() | 计算用户R(最近交易)/F(交易频率)/M(交易金额) |
| segmentUsers() | 用户分群（高价值/活跃/沉睡/流失） |
| getRFMDistribution() | RFM分布统计 |

#### T4.15.3 订单分析
| 方法 | 功能 |
|------|------|
| getOrderTrend() | 订单趋势（日/周/月） |
| getConversionRate() | 下单转化率 |
| getPeakHours() | 高峰时段分析 |
| getAverageOrderValue() | 客单价分析 |

#### T4.15.4 车辆分析
| 方法 | 功能 |
|------|------|
| getUtilizationRate() | 车辆利用率 |
| getTurnoverRate() | 周转率 |
| getRevenueRanking() | 收入排名 |
| getPopularModels() | 热门车型 |

#### T4.15.5 收入分析
| 方法 | 功能 |
|------|------|
| getRevenueTrend() | 收入趋势（同比/环比） |
| getRevenueByCategory() | 分类收入（租金/保险/服务费） |
| getRevenueForecast() | 收入预测（简单线性回归） |

#### T4.15.6 商家端Dashboard v1
| 页面 | 组件 |
|------|------|
| 数据概览 | 4个核心指标卡片 |
| 趋势图表 | 折线图（订单/收入/用户） |
| 排行榜 | Top10车辆/门店 |
