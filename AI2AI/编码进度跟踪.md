## 三、代码质量说明

### 架构亮点
1. **双Token认证**: 用户端AccessToken(7天)+RefreshToken, 商家端(2天)，黑名单机制(Redis)
2. **分布式锁**: Redis Lua脚本实现 acquireLock/releaseLock, 支持超时自动释放
3. **滑动窗口限流**: Redis ZSET 实现 rateLimit, 精确到时间窗口
4. **RBAC权限**: 元数据反射(@Roles/@Permissions) + RolesGuard 运行时校验
5. **统一异常**: HttpExceptionFilter + QueryExceptionFilter 双层异常处理, 9大类错误码
6. **软删除审计**: BaseEntity 统一 deletedAt/createdAt/updatedAt/createdBy/updatedBy

### 代码规范遵循
- TypeScript strict 模式
- JSDoc 注释（每个公开方法和类）
- kebab-case 文件名 / PascalCase 类名 / camelCase 变量
- class-validator 装饰器做 DTO 校验
- 依赖注入通过 constructor
