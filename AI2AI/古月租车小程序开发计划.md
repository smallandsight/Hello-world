# 古月租车支付宝小程序 - 3个月AI辅助开发计划

## 一、项目概述

### 1.1 项目背景
- **项目名称**: 古月租车支付宝小程序
- **开发周期**: 3个月(2026年4月1日 - 2026年6月30日)
- **开发模式**: 单人开发 + AI辅助
- **目标**: 完成核心功能开发并上线运营

### 1.2 技术栈选型

#### 前端技术栈
- **小程序框架**: uni-app (可同时发布支付宝和微信小程序)
- **UI组件库**: uni-ui + 自定义组件
- **状态管理**: Vuex
- **网络请求**: uni-request封装

#### 后端技术栈
- **运行时**: Node.js 18+
- **Web框架**: Koa2 (轻量级,适合单人开发)
- **数据库**: MySQL 8.0 (主数据库)
- **缓存**: Redis 7.0 (会话、库存缓存)
- **ORM**: Sequelize (类型安全的数据库操作)
- **认证**: JWT + 支付宝OAuth

#### 基础设施
- **云服务商**: 阿里云(与支付宝生态集成更好)
- **服务器**: ECS 2核4G起步
- **数据库**: RDS MySQL 2核4G
- **对象存储**: OSS(车辆照片、证件)
- **CDN**: 阿里云CDN(静态资源加速)
